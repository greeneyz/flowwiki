Fast & Light

The application is light and runs in the background with minimal power consumptions. It has been tested on multiple android versions.

Data Syncing

The data is synched automatically according to the filters defined in the configuraiton file.

Secure

The appplication cannot be installed by outsiders and is secured with magic link logins!

Configuration

Maintain all your settings in a single config.json file. Different configuration files can be uploaded later.

User Forms

Maintain all your settings in a single userform.json file. Different forms can be uploaded later.
